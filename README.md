TypeScript Webpack Modify Final Html Code Plugin Demo
=====================================================

如何写一个webpack的plugin，可以对最终生成的html内的代码进行修改。

```
npm install
npm run demo
```

然后看`dist`目录下生成的`index.html`文件。
